# v1.0.0 — 2026-03-26

## What's new

Initial release of MoneyTracker — a personal finance Telegram bot.

- Expense and income tracking with multi-step category selection flow
- Balance overview with income, expense, and net calculation
- Transaction history with pagination (5 per page)
- Category-based statistics with period selection (today, week, month, last week, last month, last 3 months)
- Visual bar chart breakdown per category in stats view
- Multi-currency support: USD, EUR, GBP, JPY, RUB, UAH
- Settings panel for preferred currency
- FSM-driven flows with confirmation step before saving
- Auto-registration of new users on first interaction
- Redis-backed state machine with 30-minute sliding TTL
